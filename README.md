## Development

secret values needed by the app.

    <string name="parse_app_id">PARSE APP ID HERE</string>
    <string name="parse_client_key">PARSE CLIENT KEY HERE</string>
    <string name="api_key">NORTHSTAR API KEY HERE</string>
</resources>
